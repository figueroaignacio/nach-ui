## Referencia de API

### CliCommand

| Prop         | Tipo         | Por defecto | Descripción                                                              |
| ------------ | ------------ | ----------- | ------------------------------------------------------------------------ |
| `command`    | `string`     | —           | El comando CLI que se muestra y se copia al portapapeles.                |
| `prefix`     | `string`     | `"$"`       | Símbolo que aparece antes del comando (por ej. `$`, `>`, `%`).           |
| `resetDelay` | `number`     | `2000`      | Milisegundos antes de que el estado "copiado" vuelva al ícono de copiar. |
| `onCopied`   | `() => void` | —           | Callback que se ejecuta cuando el comando se copia correctamente.        |
| `className`  | `string`     | —           | Clases CSS adicionales aplicadas al elemento raíz.                       |
